結び目の夜 — UE5ホラーゲーム

WORK 01
制作期間 約1年
担当 個人制作（企画〜実装）
エンジン Unreal Engine 5

学校のカリキュラム外でUE5を独学で習得し、授業と並行して約1年かけて完成させたホラーゲームです。 マップはプロシージャル生成で毎回異なるレイアウトを生成し、プレイヤーに飽きのこない体験を提供します。 敵・主人公・箱を開けるSEは自室で録音し、3DモデルはAI生成＋Blenderで調整。フリー素材ゼロを目標に制作しました。 主人公・朝比奈結が神隠しに遭い迷い込んだ神殿で霊に追われながら霊魂石を集め、祭壇を目指す。

技術ハイライト
  • タイル→ルーム→エリアの階層構造でマップ状態を管理するプロシージャル生成システムを自設計
  • アイテム情報を構造体ベースのデータテーブルで管理し、データとロジックを完全分離
  • GameInstanceを用いたシーン跨ぎのBGM管理システムを実装
  • Enum（TileType / Direction / RoomTileDetaile）で壁・ドア・接続方向を型安全に管理
  • カリキュラム外を独学で習得し、企画〜実装のフル工程を一人でやり切った
Unreal Engine 5 Blueprint DataTable GameInstance Procedural Generation

DirectX11 3Dアクションゲーム

WORK 02
制作期間 8ヶ月
担当 個人制作
エンジン DirectX 11

DirectX11を使用して1から構築した3Dアクションゲーム。 高ポリゴンのキャラクターと広大な草原フィールドを実装し、 初期はFPS一桁まで低下するパフォーマンス問題が発生。 ボトルネックを自分で特定し、複数の手法を組み合わせて改善した。

技術ハイライト
  • 地形処理をCPUからGPU側へ移行し、CPU-GPU間のボトルネックを解消
  • カリング・距離フォグを実装し、遠距離描画の無駄な処理負荷を削減
  • animationModel / modelRenderer / particle 等を独自クラスで設計・分離
  • ImGuiを用いたリアルタイムデバッグUIを組み込み
  • GrassInstance による大量インスタンス草描画を実装
C++ DirectX 11 HLSL ImGui Visual Studio

DirectX12 ゲームエンジン（勉強中）

WORK 03
制作期間 2ヶ月
担当 個人制作
エンジン DirectX 12

DirectX12を使用してゲームエンジンをスクラッチから構築。 DX11と比べて大幅に増えた事前準備（コマンドリスト・コマンドキュー・ディスクリプタヒープ等）を 全て手動で実装し、GPU低レイヤーの理解を深めました。

技術ハイライト
  • 頂点バッファ・インデックスバッファ・定数バッファ・シェーダーをDX12 APIで一から構築
  • デファードレンダリングパイプラインを実装（GBufferへの法線・アルベド書き込み）
  • 4段階量子化拡散＋リムライティングによるToonシェーダーをHLSLで自作
  • PSOの動的切り替えによるスカイドーム描画システムを実装
  • ImGuiによるランタイム制御UI（位置・回転・スケール・モデル選択）を組み込み
C++ DirectX 12 HLSL Deferred Rendering Toon Shading ImGui

3年次ゲーム制作 — Formula x Tactics

WORK 05
制作期間 2ヶ月
担当 メインプログラム・企画（主に）
チーム規模 8人
エンジン Unity

企画からチームをまとめてメイン担当として制作したカードゲーム。 敵・プレイヤー双方のデッキ挙動制御をメインで実装。 山札からカードを引く際の動作フロー（ドロー処理・デッキシャッフル・手札への追加）を設計し、 デッキオブジェクトが視覚的に動くアニメーション制御まで一貫して担当した。

技術ハイライト
  • デッキ・手札・捨て札の状態をクラスで分離管理し、ドロー処理を設計
  • 敵AIのデッキ挙動をプレイヤーと共通のロジックで制御し、コードを共通化
  • カードを引く際のデッキオブジェクトのアニメーション（動き・消滅）を実装
  • 山札が0枚になった際の捨て札シャッフル→再補充フローを実装
  • 企画段階からチームをまとめ、ゲームルールの設計にも主体的に関わった
Unity C# Git Card Game System Deck Control
